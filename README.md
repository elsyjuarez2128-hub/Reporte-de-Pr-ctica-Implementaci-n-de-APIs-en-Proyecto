# 📖 Introducción

Cuando se empieza a desarrollar una aplicación es muy común que empresas grandes, pequeñas o desarrolladores principiantes utilicen algo llamado **API (Application Programming Interface)**.  

Las APIs permiten que dos componentes de software puedan comunicarse entre sí mediante protocolos y definiciones establecidas. Su arquitectura suele describirse como cliente-servidor.  

Este proyecto surge como una solución tecnológica para la **UTNG**, específicamente enfocada en los alumnos que utilizan el edificio 7EEE, con el objetivo de transformar el proceso manual de solicitud de herramientas en una experiencia digital, ágil y centralizada.

---

# 💻 Descripción del Proyecto

Se desarrolló una **aplicación web modular** que permite a los alumnos:

- Solicitar herramientas del edificio 7EEE.
- Registrar datos del préstamo.
- Visualizar videos obligatorios de capacitación.
- Consultar mapas.
- Realizar pagos en caso de reposición de herramientas.

Todo el sistema está organizado por módulos para facilitar la experiencia de usuario.

---

# 🔌 APIs Implementadas

## ▶ YouTube API
Permite visualizar videos obligatorios sobre:
- Uso correcto de herramientas.
- Manejo de fuente de poder.
- Búsquedas personalizadas.

## 💳 PayPal API
Permite:
- Pago por herramientas dañadas o extraviadas.
- Apoyo en compra de nuevas herramientas.
- Uso de entorno Sandbox para pruebas.

## 🗺 Google Maps API
Muestra:
- Mapa de la universidad.
- Ubicación y alrededores.

## 📱 Hootsuite API
Permite:
- Publicar contenido en múltiples redes sociales.
- Difusión de avisos institucionales.

## 📌 Pinterest API
Permite:
- Mostrar imágenes relevantes.
- Programar contenido visual.
- Analizar métricas.

## 🎵 TikTok API
Permite:
- Visualizar contenido.
- Compartir publicaciones.
- Analizar tendencias.

## 💼 LinkedIn API
Permite:
- Compartir contenido académico.
- Iniciar sesión profesional.
- Publicar certificaciones.
- Analizar estadísticas.

# ⚙ Proceso de Integración

### 1️⃣ Registro y Autenticación
Obtención de:
- Client ID
- API Keys
- Credenciales de acceso

### 2️⃣ Configuración de Endpoints
Conexión a servicios externos como:
- PayPal Sandbox
- YouTube Data API
- Google Maps

### 3️⃣ Métodos HTTP Utilizados
- **GET** → Obtener información (videos, mapas).
- **POST** → Registrar solicitudes o publicar contenido.

### 4️⃣ Manejo de JSON
Procesamiento de respuestas para mostrar información amigable al usuario.
